Chapter 01: Input and Simulation

Overview

This chapter introduces basic input/output methods and simulation techniques used to model problems in programming contests. These techniques are often the first step toward solving computational problems that do not require deep algorithms but depend on careful implementation and attention to detail.

Core Techniques

  • Basic I/O: Using cin/cout in C++ or input()/print() in Python
  • String Parsing: Using .split(), string indexing, and conversions like int() or str()
  • Array Manipulation: Initialization, iteration, and element modification
  • Simulation: Implementing step-by-step processes based on problem descriptions

Sample Problem 1: Count Vowels

Given a string, count how many vowels it contains.

Python

def count_vowels(s):
    return sum(1 for c in s.lower() if c in "aeiou")

print(count_vowels(input()))

C++

#include <iostream>
#include <string>
using namespace std;

int main() {
    string s;
    getline(cin, s);
    int count = 0;
    for (char c : s) {
        c = tolower(c);
        if (c == 'a' || c == 'e' || c == 'i' || c == 'o' || c == 'u') count++;
    }
    cout << count << endl;
    return 0;
}

Sample Problem 2: Simulate a Robot on Grid

Given a string of commands (L, R, U, D), compute the final coordinates of a robot starting from (0, 0).

Python

x = y = 0
for c in input():
    if c == 'L': x -= 1
    elif c == 'R': x += 1
    elif c == 'U': y += 1
    elif c == 'D': y -= 1
print(x, y)

C++

#include <iostream>
using namespace std;

int main() {
    string s;
    cin >> s;
    int x = 0, y = 0;
    for (char c : s) {
        if (c == 'L') x--;
        else if (c == 'R') x++;
        else if (c == 'U') y++;
        else if (c == 'D') y--;
    }
    cout << x << " " << y << endl;
    return 0;
}
