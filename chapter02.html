Chapter 02: Sorting and Searching

Overview

This chapter covers basic sorting algorithms and binary search techniques.

Core Techniques

  • Sorting: Bubble, Selection, Insertion, Built-ins
  • Binary Search: lower_bound, upper_bound, bisect

Sample Problem

Given a list of numbers, output them in ascending order.

Python

nums = list(map(int, input().split()))
nums.sort()
print(*nums)

C++

#include 
#include 
#include 
using namespace std;
int main() {
    int n; cin >> n;
    vector a(n);
    for (int &x : a) cin >> x;
    sort(a.begin(), a.end());
    for (int x : a) cout << x << ' ';
}

Common Mistakes

Forgetting to call sort before accessing elements in sorted order.

Summary

  • Know when to use built-in sorts
  • Binary search requires sorted array
