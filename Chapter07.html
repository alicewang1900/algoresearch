Chapter 07 - Hashing & Duplicate Detection

Overview

Hash tables allow fast insertion, deletion, and lookup ‚Äî making them ideal for frequency counting and duplicate checks.

Core Techniques

  • Hash Sets for Uniqueness
  • Hash Maps for Counting

Example: Hash Set for Uniqueness (Python)

Check if any duplicates exist in the list using a set.

nums = [1, 2, 3, 1]
seen = set()
for n in nums:
    if n in seen:
        print(True)
        break
    seen.add(n)
else:
    print(False)

Example: Hash Set for Uniqueness (C++)

Use unordered_set to check for duplicates.

#include 
#include 
using namespace std;
int main() {
    unordered_set seen;
    int nums[] = {1, 2, 3, 1};
    for (int n : nums) {
        if (seen.count(n)) {
            cout << "true" << endl;
            return 0;
        }
        seen.insert(n);
    }
    cout << "false" << endl;
    return 0;
}

Example: Hash Map for Counting (Python)

Count the frequency of each element using a dictionary.

from collections import defaultdict
nums = [1, 2, 2, 3, 1]
count = defaultdict(int)
for n in nums:
    count[n] += 1
print(count)

Example: Hash Map for Counting (C++)

Use unordered_map to count frequencies.

#include 
#include 
using namespace std;
int main() {
    unordered_map count;
    int nums[] = {1, 2, 2, 3, 1};
    for (int n : nums) {
        count[n]++;
    }
    for (auto &p : count) {
        cout << p.first << ": " << p.second << endl;
    }
    return 0;
}

Error Examples & Fixes

‚ùå Hash Set Misuse

Error: Checking membership against a new empty set every time.

for n in nums:
    if n in set(): print(True)

Fix: Create the set once outside the loop.

seen = set()
for n in nums:
    if n in seen:
        print(True)
    seen.add(n)

‚ùå Hash Map Update Error

Error: Overwriting counts instead of incrementing them.

for n in nums:
    count[n] = 1

Fix: Use count[n] += 1 to accumulate.

for n in nums:
    count[n] += 1

Conclusion

Hash-based techniques are essential for efficient lookups and frequency analysis in constant time. Sets ensure uniqueness; maps support flexible counting and grouping.
