Chapter 01: Input and Simulation

Overview

This chapter introduces basic input/output methods and simulation techniques used to model problems in programming contests.

Core Techniques

  • Basic I/O: Using cin/cout or input()/print()
  • String Parsing: .split(), indexing, int(), str()
  • Array Manipulation: initialization, iteration
  • Simulation: replicating process step-by-step

Sample Problem 1: Count Vowels

Given a string, count how many vowels it contains.

def count_vowels(s):
    return sum(1 for c in s.lower() if c in "aeiou")
print(count_vowels(input()))

Sample Problem 2: Simulate Robot Movement

Given L/R/U/D command string, calculate final coordinates.

x = y = 0
for c in input():
    if c == 'L': x -= 1
    elif c == 'R': x += 1
    elif c == 'U': y += 1
    elif c == 'D': y -= 1
print(x, y)

Fix: input().split() returns a list of strings.

‚ùå Mistake 3: Infinite Loop in Simulation

# Wrong

    n = int(input())
    if n == 0:
        continue
    print(n)

Fix: Ensure valid termination condition with break.

Summary

  • Focus on edge cases & clean logic
  • Be cautious with types and conditions
  • Simulations must handle stop conditions properly
