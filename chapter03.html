Chapter 03: Prefix Sums and Difference Arrays

Overview

Learn how to use prefix sums and difference arrays to optimize range query problems.

Core Techniques

  • Prefix Sum: cumulative sum arrays
  • Difference Array: range updates in O(1)

Sample Problem

Given an array and multiple range queries, find the sum of each range efficiently.

Python

arr = list(map(int, input().split()))
pref = [0]*(len(arr)+1)
for i in range(len(arr)):
    pref[i+1] = pref[i] + arr[i]
l, r = map(int, input().split())
print(pref[r+1] - pref[l])

C++

#include 
#include 
using namespace std;
int main() {
    int n; cin >> n;
    vector a(n), p(n+1);
    for (int i = 0; i < n; i++) {

        p[i+1] = p[i] + a[i];
    }
    int l, r; cin >> l >> r;
    cout << p[r+1] - p[l];
}
